# Technology Research ERC-721 NFTs with Solidity

Technology Research for Supply Chain Updates modelled with ERC-721 NFTs in Solidity.

## Getting Started

### Prerequisites

  * `node v12.13.x` or later
  * Truffle v5.1.64

### Running locally

  * Clone the repo. 
  * Run `npm install`
  * Run `truffle migrate`
